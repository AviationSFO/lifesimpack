# lifesimpack
## Description
Life 'simulation' package for python created as a joke.

## How to use
import:
```shell
import lifesim
```
create person:
```shell
person_name = lifesim.Person("First Last", height, weight, age (0 is default), status (ignorable))
```
# Installation
```shell
pip install lifesimpack-AviationSFO
```
